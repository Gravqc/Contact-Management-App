# Rest API Convention:
- Get all contacts: /api/contacts   (GET)
- Get Contact: /api/contacts/:id    (GET)
- Create Contact: /api/contacts     (POST)
- Update Contact: /api/contacts/:id (PUT)
- Delete Contact: /api/contact/:id  (DELETE)
